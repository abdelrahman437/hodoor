# Hodoor - حضور وانصراف بالبصمة

تطبيق أندرويد بسيط لتسجيل حضور وانصراف الموظفين على نفس الموبايل باستخدام بصمة الجهاز.

## المميزات

- تسجيل الحضور بالبصمة.
- تسجيل الانصراف بالبصمة.
- إدخال:
  - بداية الدوام.
  - نهاية الدوام.
  - الحد الأدنى للساعات قبل احتساب الأوفر تايم.
- حساب الأوفر تايم بالطريقتين معًا (الأكبر بينهما):
  1. بعد نهاية الدوام الرسمي.
  2. بعد تجاوز عدد ساعات معين.
- حفظ السجلات محليًا على الجهاز (Room Database).

## معادلة الأوفر تايم

عند الانصراف:

- `overtimeByHours = max(0, workedHours - overtimeAfterHours)`
- `overtimeByEndTime = max(0, now - shiftEnd)`
- `finalOvertime = max(overtimeByHours, overtimeByEndTime)`

## المتطلبات

- Android Studio حديث (JDK 17).
- Android SDK 34.

## التشغيل

1. افتح المشروع من Android Studio.
2. اسمح بعمل Gradle Sync.
3. شغل التطبيق على جهاز يدعم البصمة.

## ملاحظات

- حاليًا البيانات محلية على نفس الجهاز.
- لو حابب نسخة لفريق كامل بقاعدة مشتركة، نقدر نضيف Firebase.
